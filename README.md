# ToDoDoesDid



## 💁🏻‍♀️ Introduce
ToDoDoesDid는 '일정'을 주제로 유저가 달력에 일정을 추가하면 해당 날짜에 일정을 등록할 수 있다. 현재 메인 페이지까지 완료되었으며, React 라이브러리를 사용해 각각의 기능들을 구현했고 일정을 서버에 저장, 서버로부터 일정을 받아올 수 있다. 추후에 로그인 기능과 중요한 일정만 따로 값을 받아오며 원하는 일정을 검색하는 기능을 구현 예정이다.
